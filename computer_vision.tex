Visão computacional, em linhas gerais, é a tecnologia capaz de transformar imagens do mundo real em uma representação que computadores são capazes de interpretar e processar. Muitos definem visão computacional como a ciência das máquinas que enxergam. É uma base teórica e tecnológica para a construção de sistemas artificiais que obtém informações de imagens ou quaisquer dados multidimensionais. Mas como os computadores podem entender o mundo visual dos humanos? Quais vantagens e quais tipos de aplicações podem existir a partir desse conceito?
